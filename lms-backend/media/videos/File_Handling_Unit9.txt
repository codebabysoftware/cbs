Topic 9: File Management Operations**



9.1 Introduction**

Python provides modules to manage files at the **operating system level**:

* `os` → Interact with the file system (rename, delete, check existence)
* `shutil` → High-level operations (copy, move)

---

9.2 Importing Required Modules**

```python
import os
import shutil
```

---

9.3 Checking if a File Exists**

```python
if os.path.exists("data.txt"):
    print("File exists")
else:
    print("File does not exist")
```

* Returns `True` if file is present

---

9.4 Deleting a File**

Program 1: Delete a File**

```python
if os.path.exists("data.txt"):
    os.remove("data.txt")
    print("File deleted")
else:
    print("File does not exist")
```

* `os.remove()` deletes a file
* Raises `FileNotFoundError` if file does not exist

---

9.5 Renaming a File**

Program 2: Rename a File**

```python
if os.path.exists("oldname.txt"):
    os.rename("oldname.txt", "newname.txt")
    print("File renamed")
```

* Changes file name without moving location

---

9.6 Copying Files Using `shutil`**

Program 3: Copy a File**

```python
shutil.copy("source.txt", "destination.txt")
```

* Copies content and preserves permissions
* Does not copy metadata (use `copy2()` to preserve metadata)

```python
shutil.copy2("source.txt", "destination.txt")
```

---

9.7 Moving Files Using `shutil`**

Program 4: Move a File**

```python
shutil.move("file.txt", "folder/file.txt")
```

* Moves file to a new location
* Can also rename in the process

---

9.8 Creating Directories**

Program 5: Create Directory**

```python
os.mkdir("new_folder")  # Single directory
os.makedirs("folder1/folder2")  # Nested directories
```

---

9.9 Deleting Directories**

```python
os.rmdir("new_folder")  # Remove empty directory
shutil.rmtree("folder1")  # Remove non-empty directory
```

---

9.10 Listing Files in a Directory**

```python
files = os.listdir(".")  # List files in current directory
print(files)
```

* Returns a list of file and folder names

---

9.11 Checking File Type**

```python
if os.path.isfile("data.txt"):
    print("It is a file")
if os.path.isdir("folder1"):
    print("It is a folder")
```

* Useful for distinguishing files and directories

---

9.12 Getting File Size**

```python
size = os.path.getsize("data.txt")
print(f"File size: {size} bytes")
```

* Returns size in bytes

---

9.13 Common Use Cases**

* Organizing files into folders
* Automating backups
* Renaming files in bulk
* Cleaning temporary files

---

9.14 Best Practices**

1. Always check if a file exists before deleting or renaming
2. Use `shutil.copy2()` to preserve metadata
3. Avoid `os.rmdir()` on non-empty folders; use `shutil.rmtree()`
4. Handle exceptions with `try/except` for safer file operations

---

9.15 Exam-Oriented Points**

* `os.remove()` → delete file
* `os.rename()` → rename file
* `shutil.copy()` → copy file
* `shutil.move()` → move/rename file
* `os.listdir()` → list directory contents
